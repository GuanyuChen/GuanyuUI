# WebModules
# HTML5实现的移动端数据可视化

元素：

1. 移动端
2. 可视化
3. CANVAS
4. 数据报告

## 产品设计

1. 首页载入动画
2. 滑动切换页面
3. 具体页面
    + 首页
    + 图文
    + 折线图
    + 饼图
    + 柱图
    + 雷达图
    + 环图
    + 散点图
    + 尾页

## 技术方案设计

1. 页面DOM操作采用`jQuery` 
2. 页面切换功能采用`FullPage.js`(一款基于jQuery的插件，简单、易用、跨浏览器)
3. 页面组织方式采用Page--Component(Page就是页面切换的每一页，可以理解为一个满屏的DIV，Component是页面上应用的组件，简单分为图文组件和图表组件)
